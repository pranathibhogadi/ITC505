In this lab exercise, I solved the problem of sorting an array of integers upon clicking a button. I implemented this functionality using JavaScript. Upon clicking the "Sort Array" button, the script sorts the array in ascending order and displays the sorted array in the output area.

To achieve this, I used event listeners to detect clicks on the button and then sorted the array using the built-in `sort()` method in JavaScript. Finally, I displayed the sorted array in the output area.
